# WaterCharges

**Commercial water charges:**
0 - 10KL -> Rs.50 per KL
10 - 25KL -> Rs.57 per KL
25 - 50KL -> Rs.65 per KL
50 - 75KL -> Rs.76 per KL
75KL >>  -> Rs.87 per KL

**Domestic Water Charges**
0 - 8KL -> Rs.7 per KL
8 - 25KL -> Rs.11 per KL
25 - 50KL -> Rs.26 per KL
50KL >>  -> Rs.45 per KL

Bill = Water Charges + 25% Sanitary Charges + Meter Charges(150) + Borewell Charges(250) + Other Charges
